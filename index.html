Proyectos

Penca Copa América - PHP

Proyecto académico que permite a los usuarios crear pencas y a un administrador ingresar los resultados de los partidos. El sistema calcula automáticamente los puntos obtenidos por cada participante en función de los aciertos.

Tecnologías utilizadas: PHP, MySQL, HTML, CSS

GitHub

Plataforma turismo.uy - Java

Plataforma de gestión social de actividades turísticas que facilita la interacción entre turistas y proveedores. Permite el registro de usuarios, creación de actividades turísticas, organización de salidas y comercialización de paquetes promocionales.

El sistema incluye un Servidor Central y una Estación de Trabajo (Administrador), desarrollados en Java con interfaz Swing. En la segunda iteración, se añadió un Componente Web para acceso desde navegador, autenticación de usuarios (visitantes, turistas, proveedores) y nuevas funcionalidades como gestión de categorías y confirmación de actividades por el administrador.

Funcionalidades clave:

Tecnologías utilizadas: Java, Swing, MySQL, JPA, JSP

GitHub

Sistema de Cobro Automatizado de Peaje - Jackarta EE

Sistema diseñado para gestionar el cobro automatizado de peajes, basado en el funcionamiento de la empresa Telepeaje. Permite la identificación de vehículos mediante Tag (dispositivo electrónico) o matrícula, y ofrece diferentes métodos de pago para usuarios uruguayos y extranjeros.

Tecnologías involucradas: Jakarta EE, WildFly y Jakarta Persistence API.

GitHub

Gestión de Ferretería "Tu Ferretería" - Flutter

Aplicación móvil desarrollada específicamente para la gestión de la ferretería "Tu Ferretería". La aplicación sigue una arquitectura distribuida, con persistencia y backend alojados en un servidor externo.

Tecnologías involucradas: Flutter (Android/iOS) y Firebase.

GitHub

Gestión de Flujo de Clientes en Oficinas Comerciales - .NET

Aplicación desarrollada con .NET 8 para gestionar el flujo de público en oficinas de atención al cliente de alto tránsito, distribuidas a nivel nacional. El sistema facilita la operativa diaria y brinda información estadística para la mejora continua del servicio.

Tecnologías utilizadas: Blazor, .NET 8 y EntityFramwork

GitHub
